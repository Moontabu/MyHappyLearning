<!--⦁	Procedural Content Generation  -->

⦁   过程内容生成

<!--⦁	Introduction to PCG  -->
⦁   PCG概论

<!--Procedural content generation(PCG)[5] in video games is any content that has been produced by the computer through an algorithm. It has many applications besides video games, but the following are some of the applications and benefits in video games: --> 
电子游戏中的程序内容生成(PCG)[5]是指计算机通过算法生成的任何内容。除了电子游戏，它还有许多应用，以下是电子游戏中的一些应用和好处:


<!--⦁	Replayability: -->  
⦁   多重使用方法
<!--Using procedural generation we can create infinite levels, a game that generate dungeons procedurally will offer a different experience any time we play. Games like “Diablo” or “Rogue” started using this techniques.  -->
使用程序生成，我们可以创造无限的关卡，以程序生成地下城的游戏将在我们玩的任何时候提供不同的体验。《暗黑破坏神》或《Rogue(不会翻译这个）》等游戏开始使用这种技术。  

<!--⦁	Terrain creation:   -->
⦁   地形创造

<!--As explained previously we can generate heightmaps procedurally and use them to create terrains, this way the process is automatic. We can also compute the height value of the terrain at runtime and create an infinite terrain. Games like “No man’s sky” uses this technique.  -->
如前所述，我们可以生成高度地图，并使用它们来创建地形，这样这个过程是自动的。我们也可以在运行时计算地形的高度值，并创建一个无限地形。《无人深空》等游戏就使用了这种技术。

<!--⦁	Optimize memory usage:   -->

⦁   优化内存使用
<!--Following the previous example, computing things like terrain data at runtime, it will also minimize the memory usage since we won’t need to store that anywhere.  -->
按照前面的例子，在运行时计算地形数据之类的东西，它还将最小化内存使用量，因为我们不需要将它存储在任何地方。
<!--⦁	Speed up design process:  --> 
⦁  加快设计进程
<!--We can also place assets procedurally, for example grass, trees, houses, … Or even generate cities procedurally. There is no need to mention the time that could be saved using this techniques.  -->
我们还可以程序化地放置资产，如草地、树木、房屋等，甚至是程序化地生成城市。没有必要提及使用这种技术可以节省的时间。

<!--⦁	Texture generation:-->
⦁   纹理生成
<!--We can generate textures procedurally by combining noises and filters to generate PBR textures. One of its main strength is that since it is calculated through an algorithm, procedural textures are resolution independent. They will not lose detail when scaling them. They are also really customizable since a change of one parameter can dramatically change the final result.-->
我们可以通过结合噪声和滤波器来生成纹理。它的一个主要优势是，由于它是通过算法计算的，程序纹理是分辨率独立的。当扩展它们时，它们不会丢失细节。它们实际上是可定制的，因为更改一个参数可能会极大地改变最终结果。
